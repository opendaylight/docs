========
L2Switch
========

Overview
========
The OpenDaylight (ODL) L2Switch project provides core Layer 2 switching functionality by managing address learning and
loop prevention within a software-defined network. It leverages the OpenFlow protocol to dynamically populate flow
tables, allowing for seamless packet forwarding and network topology discovery across an ODL-managed controller.
