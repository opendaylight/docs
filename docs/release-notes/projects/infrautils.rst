==========
Infrautils
==========

Infrautils project provides low level utilities for use by other OpenDaylight projects, including:

* @Inject DI
* Utils incl. org.opendaylight.infrautils.utils.concurrent
* Test Utilities
* Job Coordinator
* Ready Service
* Integration Test Utilities (itestutils)
* Caches
* Diagstatus
* Metrics
