==========
InfraUtils
==========

Overview
========

The InfraUtils project provides a low-level utility for use by other OpenDaylight projects, including:

* @Inject DI
* ``Utils`` incl. ``org.opendaylight.infrautils.utils.concurrent``
* Test Utilities
* Ready Service
* Integration Test Utilities (``itestutils``)
* ``Diagstatus``
