====
gNMI
====

Overview
========
gNMI plugin implementation which provides:

* capabilities to manage network of gNMI capable devices
* ability to users to manage this network with RESTCONF as a northbound
* possibility to use this plugin within other projects such like TPCE

Behavior/Feature Changes
========================
There are no deprecated or removed features.

The initial project release - version 1.0.0 brings original `lighty.io <https://lighty.io/>`_
based implementation to ODL environment.

New Features
============
There are no deprecated or removed features. Functionality is the same as in
`lighty.io 22.1.0 <https://github.com/PANTHEONtech/lighty/releases/tag/22.1.0>`_ release.

Deprecated and Removed Features
===============================
There are no deprecated or removed features. Functionality is the same as in
`lighty.io 22.1.0 <https://github.com/PANTHEONtech/lighty/releases/tag/22.1.0>`_ release.
