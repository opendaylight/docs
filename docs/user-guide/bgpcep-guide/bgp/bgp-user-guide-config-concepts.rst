BGP RIB API
^^^^^^^^^^^
This tree illustrates the BGP RIBs organization in datastore.

.. code-block:: console

    bgp-rib
      +--ro rib* [id]
         +--ro id         rib-id
         +--ro peer* [peer-id]
         |  +--ro peer-id                  peer-id
         |  +--ro peer-role                peer-role
         |  +--ro simple-routing-policy?   simple-routing-policy
         |  +--ro supported-tables* [afi safi]
         |  |  +--ro afi             identityref
         |  |  +--ro safi            identityref
         |  |  +--ro send-receive?   send-receive
         |  +--ro adj-rib-in
         |  |  +--ro tables* [afi safi]
         |  |     +--ro afi           identityref
         |  |     +--ro safi          identityref
         |  |     +--ro attributes
         |  |     |  +--ro uptodate?   boolean
         |  |     +--ro (routes)?
         |  +--ro effective-rib-in
         |  |  +--ro tables* [afi safi]
         |  |     +--ro afi           identityref
         |  |     +--ro safi          identityref
         |  |     +--ro attributes
         |  |     |  +--ro uptodate?   boolean
         |  |     +--ro (routes)?
         |  +--ro adj-rib-out
         |     +--ro tables* [afi safi]
         |        +--ro afi           identityref
         |        +--ro safi          identityref
         |        +--ro attributes
         |        |  +--ro uptodate?   boolean
         |        +--ro (routes)?
         +--ro loc-rib
            +--ro tables* [afi safi]
               +--ro afi           identityref
               +--ro safi          identityref
               +--ro attributes
               |  +--ro uptodate?   boolean
               +--ro (routes)?
